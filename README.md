# Atlas Map Generator & History Simulator

## Building

Debug mode:

```sh
cargo build
```

Release mode:

```sh
cargo build --release
```

## Running

With source code:

```sh
cargo run --bin=atlas_gen
# or
cargo run --bin=atlas_sim
```

With precompiled binary:

```sh
./atlas_gen.exe
# or 
./atlas_sim.exe
```
